# 📰 Fake News Detector  

A machine learning–powered web application that identifies whether a news article is **real or fake** using **natural language processing (NLP)** techniques.  
Built to promote digital literacy and help users verify the credibility of online information.  

---

## 🌟 Features  
- 🔍 Detects fake news from user-input text or URLs.  
- 🧠 Uses NLP preprocessing and ML models for classification.  
- 📊 Displays prediction probability and explanation for transparency.  
- 🌐 Simple and intuitive web interface.  

---

## 🧩 Tech Stack  
**Frontend:** HTML, CSS, JavaScript  
**Backend:** Python (Flask)  
**Libraries:** scikit-learn, NLTK, pandas, NumPy  
**Model:** Logistic Regression / Naive Bayes / (your model name)  
**Deployment:** Heroku / Docker / Localhost
